# Cadastro de Pacientes

Este projeto é uma aplicação React desenvolvida para cadastrar informações de pacientes, com interface simples e visual organizado.

## Sobre o projeto

A aplicação permite:
- inserir nome, idade, cidade, sexo e tipo de atendimento;
- visualizar o horário atual na tela;
- enviar os dados por meio de um formulário.

## Tecnologias utilizadas

- React
- JavaScript
- CSS moderno para estilização

## Como executar

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie a aplicação:
   ```bash
   npm start
   ```
3. Abra o navegador em:
   ```text
   http://localhost:3000
   ```

## Scripts disponíveis

### `npm start`
Executa a aplicação em modo de desenvolvimento.

### `npm test`
Executa os testes da aplicação.

### `npm run build`
Gera a versão otimizada para produção.

## Estrutura principal

- `src/` — código fonte da aplicação
- `src/components/` — componentes reutilizáveis
- `src/App.js` — componente principal
- `src/App.css` — estilos da interface
